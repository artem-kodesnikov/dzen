# DZEN

- [DEMO LINK]('https://dzen-test-4gcn.onrender.com')

Technologies used:
  - React.js, React-router
  - Redux-Toolkit
  - Typescript
  - Module CSS, React Bootstrap, react-transiton-group

### Run project
- clone the repo
- npm install
- npm start

### Available to use
Orders:
- Delete order
- Open or close order products
- Add(static data)to order or delete product 

Products:
- List of all product
- Filter select
